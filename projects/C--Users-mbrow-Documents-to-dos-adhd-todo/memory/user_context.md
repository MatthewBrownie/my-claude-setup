description: Who the user is and why they built this app
type: user

The user has ADHD and built this app specifically for their own daily use. Features are motivated by real ADHD challenges — time blindness (prompted the work timer and "before 5pm" banner), out-of-sight/out-of-mind (prompted always-visible sidebar), and task overwhelm (prompted focus mode). They mentioned a puppy as a concrete example of how time blindness manifests (stepping away "briefly" and losing an hour).

Frame suggestions in terms of ADHD usability, not general productivity.
